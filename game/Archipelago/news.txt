=== IMPORTANT: CHARACTER SWITCHING ===
IF YOU WANT TO CREATE A NEW CHARACTER AFTER PLAYING ANOTHER ONE,
CLOSE THE GAME COMPLETELY AND RESTART IT BEFORE CREATING A NEW CHARACTER.
DO NOT SWITCH CHARACTERS FROM THE CHARACTER SELECT SCREEN.

=== IMPORTANT: INSTALLATION ===
The launcher requires you to point to your existing Diablo II + Lord of
Destruction installation. Original game files are copied from your own
install automatically. Only Diablo II Classic is supported (NOT Resurrected).
